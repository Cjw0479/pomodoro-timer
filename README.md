# Pomodoro Timer

A simple, single-file Pomodoro timer built with vanilla HTML, CSS, and JavaScript. No dependencies, no build step — just open in a browser.

## Features

- 25 min focus / 5 min short break / 15 min long break
- Circular progress ring
- Auto-advances between phases with an audio beep
- Tracks completed pomodoros per session
- Cycle dots show progress toward a long break

## How to use

Open `index.html` in any browser, or visit the live deployment.

## Tech

Vanilla HTML/CSS/JS — Web Audio API for the beep, SVG for the progress ring.
